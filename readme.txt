== Description ==

A plugin for show wooCommerce products on single page and customers can quick order from mobile.

<strong>Features are coverd:</strong>

<ul>
<li>Faster and easier way to order from online shop by mobile.</li>
<li>Fully responsive for any device.</li>
<li>Category wise product search.</li>
<li>Thirty (30) can display Cart images from admin.</li>
<li>Eight (8) different mini-cart templates.</li>
<li>User can display or hide full image preview from admin.</li>
<li>User can set Shop Stock Condition: Open, Lunch time, Home delivery only, Closed with custom message.</li>
<li>Search button background, border and text color can be set from admin.</li>
<li>Menu and Sub Menu background hover and text color can be set.</li>
<li>Item name, hover and description color can be set.</li>
</ul>

== Installation ==

The plugin can be installed like any other.

1. Log into WordPress
1. Select Plugins, then "Add New"
1. Select Upload Plugin
1. Choose the zip file, then Install and activate

Once installed, you will need to create your first survey by going to RCH Menu Cart for WooCommerce.
